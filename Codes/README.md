# CaneAI Intelligence System

This folder contains the core implementation of the CaneAI system, including the main application and supporting development scripts.

The system focuses on AI-based sugarcane quality assessment using sensor inputs such as IR value, dielectric value, and moisture content.

## Features

- Real-time sensor data processing
- AI-based Brix prediction
- Automated decision (Accept / Hold / Reject)
- Interactive dashboard interface
- Supporting scripts used during model development and testing

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Arduino

## System Flow

Sensor → Arduino → AI Model → Decision → UI
