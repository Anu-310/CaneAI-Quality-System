# CaneAI Intelligence System

AI-powered sugarcane quality monitoring system using real-time sensor inputs.

## Features
- Real-time sensor data processing
- AI-based Brix prediction
- Automated decision (Accept / Hold / Reject)
- Industrial dashboard interface

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Arduino

## System Flow
Sensor → Arduino → AI Model → Decision → UI
